<h1>Mouse Cursor Colour Picker</h1>

<p>A lightweight Python utility that tracks your mouse cursor in real-time and displays the current pixel's RGB and HEX values directly in your terminal.</p>

<h2>Features</h2>

- Real-time Tracking: Updates the colour values every 100 milliseconds.

- Robust Conversion: Handles standard RGB and RGBA (Alpha channel) inputs gracefully.

- Performance Optimised: Uses a 1x1 pixel "snapshot" method to minimise CPU and memory usage.

- Clean Output: Uses carriage returns to provide a live, single-line update in the console.
